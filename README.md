# 🏦 Mini Banking System

A simple **Python-based Mini Banking System** developed using basic Python concepts such as:
- Lists
- Functions
- Loops
- Conditionals
- User Input/Output

This project is beginner-friendly and demonstrates how a basic banking application works using console-based interaction.

---

## 📌 Features

✅ Create New Account  
✅ Secure Login using Account Number and PIN  
✅ Deposit Money  
✅ Withdraw Money  
✅ Check Account Balance  
✅ View Recent Transactions  
✅ Logout System  
✅ Automatic Account Number Generation  

---

## 🛠️ Technologies Used

- Python 3
- Basic Data Structures (Lists & Tuples)
- Functions
- Conditional Statements
- Loops
